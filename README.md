# Garden Shed Rentals

A modern Next.js site for renting garden sheds. Flexible rental terms, multiple shed sizes, and professional setup/removal.

## Features

- Rent garden sheds (short-term or seasonal)
- Quick quote form for easy booking
- Responsive design & optimized images
- Accessible, clear layout
- Simple static owner contact info

---

## Getting Started

1. `npm install`
2. `npm run dev`
3. Visit [http://localhost:3000](http://localhost:3000)
